# Triage Labels

This file maps the triage roles used by agent workflows to this repo's GitHub
labels.

| Triage role        | GitHub label       | Meaning                                  |
| ------------------ | ------------------ | ---------------------------------------- |
| `needs-triage`     | `needs-triage`     | Maintainer needs to evaluate this issue  |
| `needs-info`       | `needs-info`       | Waiting on reporter for more information |
| `ready-for-agent`  | `ready-for-agent`  | Fully specified, ready for agent work    |
| `ready-for-human`  | `ready-for-human`  | Requires human implementation            |
| `wontfix`          | `wontfix`          | Will not be actioned                     |

When assigning triage state, use the corresponding label string from this table.
